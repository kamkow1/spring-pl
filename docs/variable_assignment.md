# variable assignment

## create a variable

```text
declare 123 -> $num;

declare "hello" -> $str;

-53.53 -> $flt;

True -> $bl;

Null -> $n;
```

to create an empty varible, instanctiate it with null. you can alway re assign the variable later.

## re assign the variable

```text
declare 6378 -> $num;

562 -> $num;

println $num; -- prints 562
```

## general syntax

```text
declare expression -> $var_name;

expression -> $var_name;
```
